omega uses cpu runtime detection so should work on all x86_64 cpu and is tuned to use latest featured cpu's as well.

* a x86_64-v3 or x86_64-v4 specific build is not needed for cachyos in this case.

* This package should work on cachyos and arch.



First using 'yay' install openarena from aur:
```
yay -si openarena
```
This will install standard openarena 0.8.8-4 to /opt/openarena

Then install any release package from releases here:
```
sudo pacman -si ./pkgname
```
